# Mental Health Companion Chatbot

This project is a simple Streamlit-based chatbot designed to support student mental well-being.  
It identifies the user's emotional state from their text input and provides supportive, empathetic responses.

## Features

- Detects basic emotions from text (happy, sad, stress, anger, neutral)
- Provides supportive responses based on emotion
- Simple and clean Streamlit interface
- Works instantly in the browser
- Beginner-friendly implementation

## Technologies Used

- Python
- Streamlit
- Basic rule-based emotion detection
- Git and GitHub for version control

## How It Works

1. User enters text describing how they feel.
2. The system checks for keywords to detect the emotion.
3. A predefined supportive message is generated.
4. The result is shown in the web interface.
