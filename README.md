# Mental Health Companion Chatbot

This project is a simple Streamlit-based chatbot that supports student mental well-being.  
The application identifies the emotional tone in the user's input text and provides a supportive response based on the detected emotion.

## Features
- Basic emotion detection (happy, sad, stress, anger, neutral)
- Simple rule-based text analysis
- Clean and easy-to-use Streamlit interface
- Works in the browser without installation (via Streamlit Cloud)

## Technologies Used
- Python
- Streamlit
- Basic text matching for emotion detection
- Git and GitHub for version control

## Project Structure
mental-health-chatbot/
│
├── app.py          # Main application file
└── README.md       # Project documentation

## How It Works
1. The user enters a sentence describing how they feel.
2. The application checks the text for emotion-related keywords.
3. Based on the detected emotion, a suitable supportive response is displayed.
4. Everything runs on a simple Streamlit web interface.
